一.获取手机验证码
接口链接：
http://localhost/chutao/index.php/api/user/getphonecode?phone=18651551916&optype=1

接口参数：
phone : 手机号
optype : 操作类型（1：注册时获取验证码；2：重置密码时候获取验证码）

返回值：
"errresult":true|false 执行成功返回true，执行失败返回false
datas : 
    phonenum->号码 
    code->验证码  
    optype:1->注册时获取的验证码 
    2->重置密码时候获取验证码
